# Horde Defence

🎮 **[Play it here → https://esoinila.github.io/HordeDefence/](https://esoinila.github.io/HordeDefence/)**

A browser-based tower-defence / horde-wave game built with ASP.NET Core and Canvas.

## How to play

1. During **Entrenchment** phase, spend Supplies to place defences on the grid (Trenches, Wire, Maxim Guns, Moats, etc.).
2. Press **🚀 LAUNCH HORDE** to start the wave.
3. Survive! Channel the horde through your kill-zones and protect the Castle Keep in the centre.
4. After each wave you can replay, modify, or advance to the next (x1.5 difficulty) wave.

## Running locally

```bash
dotnet run
```

Then open `http://localhost:5000` in your browser.
